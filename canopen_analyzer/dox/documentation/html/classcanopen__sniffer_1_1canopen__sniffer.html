Detailed Description

CANopen bus sniffer thread.

The sniffer opens a socketcan interface, receives can.Message frames, enqueues them on raw_frame for downstream processing, and optionally writes raw frames to an export file for offline analysis. The thread supports a graceful shutdown via stop(). Logging is performed on a per-instance logger.

◆ run()

def run (   self)

Main loop of the sniffer thread. 

   @details
   Continuously receives frames from the CAN bus using a short timeout,
   handles interrupt-like exceptions gracefully, and delegates message
   processing to `handle_received_message`. On exit, export file and bus resources
   are closed/shutdown cleanly.
